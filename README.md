# Newsletter Signup Form

A responsive newsletter signup form with email validation.

**Live Demo:** https://hasby-shanessa.github.io/news_letter_webDev/

## Features

- Email validation
- Success modal on valid submission
- Error state for invalid emails
- Responsive design (desktop & mobile)
- Hover effects on buttons

## Files

```
news_letter_webDev/
├── index.html    → Page structure
├── styles.css    → All styling
└── script.js     → Form validation
```

## Email Validation Rules

- Must contain @ symbol
- Must have text before @
- Must have a dot after @
- Domain must be at least 2 characters
- Extension must be at least 2 characters

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
